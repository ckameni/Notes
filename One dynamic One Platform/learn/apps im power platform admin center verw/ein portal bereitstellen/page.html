Um ein Portal mit Microsoft Power Pages bereitzustellen und zu verwalten, befolgen Sie diese Schritte:
Websites verwalten:
  • Mandantenansicht:
    • Gehen Sie zum Power Platform Admin Center.
    • Wählen Sie im linken Navigationsbereich den Link Ressourcen > Power Pages-Websites aus.
    • Hier können Sie alle Power Pages-Websites in Ihrem Mandanten verwalten.
  • Umgebungsansicht:
    • Wählen Sie die gewünschte Umgebung aus.
    • Im Bereich der Power Pages-Websites können Sie alle Websites verwalten, die innerhalb dieser Umgebung erstellt wurden.
Wichtige Funktionen:
  • Power Pages ermöglicht es, Websites zu erstellen, bei denen Benutzer mit verschiedenen Identitäten (einschließlich externer Benutzer) auf Daten im Microsoft Dataverse zugreifen können.
  • Sie können Benutzer erlauben, Daten zu erstellen, anzuzeigen oder sogar Inhalte anonym zu durchsuchen.
  • Als Administrator können Sie diese Websites sowohl auf Mandanten- als auch Umgebungsniveau verwalten.

Dies ermöglicht es, sowohl intern als auch extern eine Vielzahl von Anwendungen und Inhalten zu bieten, ohne tiefgehende Entwicklungskenntnisse zu benötigen.
 

Um Power Pages-Websites zu verwalten, gehen Sie folgendermaßen vor:

1. Zugang zum Power Platform Admin Center:
    • Melden Sie sich im Power Platform Admin Center an.
    • Gehen Sie zu Ressourcen > Power Pages-Websites im linken Navigationsbereich, um eine Übersicht über alle Websites zu erhalten.
    • Alternativ können Sie die Umgebung auswählen, in der die Website erstellt wurde, und dann Power Pages-Websites aus dem Bereich Ressourcen dieser Umgebung auswählen.

2. Verwalten einer Power Pages-Website:
    • Wählen Sie die gewünschte Power Pages-Website
  •   In der Befehlsleiste oben auf dem Bildschirm werden verschiedene Optionen angezeigt,
    • Governancesteuerelemente:
    • Mit dieser Option können Sie den anonymen Zugriff auf Ihre Website(n) deaktivieren, um die Kontrolle darüber zu behalten, wer auf die Inhalte zugreifen kann.
    • Verwalten:
    • Hier können Sie verschiedene Aspekte der Website verwalten, wie etwa Einstellungen, Design, Benutzerverwaltung und mehr. Weitere Details dazu finden Sie in den spezifischen
    • Portalpaket(e) auf Upgrade prüfen:
    • Diese Option ermöglicht es Ihnen, nach verfügbaren Upgrades für Ihre Portalpakete zu suchen und diese zu
    • Cache leeren:
    • Mit dieser Funktion können Sie den serverseitigen Cache leeren, was hilfreich ist, wenn Änderungen an der Website vorgenommen wurden und Sie sicherstellen möchten, dass die neuesten Inhalte angezeigt werden.

Wichtige Hinweise:
  • Governancesteuerelemente sind besonders nützlich, wenn Sie den Zugriff auf Ihre Power Pages-Website steuern möchten, insbesondere in Bezug auf anonymen Zugriff.
  • Cache leeren ist ein wichtiges Tool zur Fehlerbehebung und zum Aktualisieren der Inhalte auf Ihrer Website.

Diese Optionen helfen Ihnen dabei, die Verwaltung Ihrer Power Pages-Websites effizient zu gestalten.
