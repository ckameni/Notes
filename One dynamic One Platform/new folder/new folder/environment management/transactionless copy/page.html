Table group metadata is used to dynamically determine the list of tables that will not be copied to your target environment. By default, the following table groups are excluded:
  • Transaction
  • TransactionHeader
  • TransactionLine
  • Worksheet
  • WorksheetHeader
  • WorksheetLine
  • Staging

These tables will not be copied to the target environment.
If you are an ISV or a customer with custom tables you have created, please review this property and update it as needed. When using this feature, consider whether you want your custom tables to be copied or not, and set this property accordingly for your custom tables.
