Nmap | Seven Must Know Techniques in Seven Minutes 

1- Default scan
> nmap 10.0.2.13



2- OS Enumeration:( What operating system are installed on the devices we are scanning)
You can scan an entire subnet with the expression: nmap 10.0.2.0/13
> nmap 10.0.2.0/13 -o



3- Pin Scan( Excellent for network discovery, when you want to know which devices are on the network)
> nmap -sP 10.0.2.0/24


4- (Scan to see if a specific port is open for a device)
> namp -p 22 10.0.2.13

if you don't know the port, you can enter the service
> nmap ssh 10.0.2.13
> nmap ftp 10.0.2.13





5- How to lookup the version of the service that is running on certain devices
> nmap -sV 10.0.2.13





6- Scan using all Flag
> nmap -A 10.0.2.13


7- Scan Using Nmap scripts
> nmap -script vuln 10.0.2.13
