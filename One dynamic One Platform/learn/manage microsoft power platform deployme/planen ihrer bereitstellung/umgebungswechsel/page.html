Umgebungswechsel
Es ist wichtig, Ihre Implementierung gut zu planen, aber Sie können jederzeit den Umgebungszweck von Produktion auf  Sandbox zu ändern. Und umgekehr..


Wenn Sie vor der Installation einer Lösung eine Sicherung Ihrer Produktionsumgebung erstellt haben und nach der Installation Probleme auftreten oder die Lösung nicht
entfernt werden kann, können Sie die Umgebung aus der Sicherung wiederherstellen. Beachten Sie jedoch, dass Sicherungen nicht direkt in einer Produktionsumgebung
 wiederhergestellt werden können. Die Umgebung muss zuerst auf Sandbox umgestellt werden. Danach können Sie die Wiederherstellung durchführen und anschließend
wieder auf Produktion wechseln. Diese Einschränkung verhindert, dass Ihre Produktionsumgebung versehentlich überschrieben wird.
