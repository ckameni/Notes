Automate Environment creation

Ex1: Creates a new trial environment in the United States with the display name 'HQ Apps'  
New-AdminPowerAppEnvironment -DisplayName 'HQ Apps' -Location unitedstates -EnvironmentSku Trial

Ex2: Creates a new production environment in the United States with the display name 'HQ Dev'  
New-AdminPowerAppEnvironment -DisplayName 'HQ Dev' -Location unitedstates -EnvironmentSku Production


Ex2: Creates a new production environment in the United States with the display name 'HQ Dev'  
New-AdminPowerAppEnvironment -DisplayName 'NewSandbox' -LocationName "UnitedStates" -EnvironmentSku Sandbox -ProvisionDatabase -Templates 'D365_FinOps_Finance'
