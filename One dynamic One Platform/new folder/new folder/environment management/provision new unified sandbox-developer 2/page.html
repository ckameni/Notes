Acquire License:
 Visit Marketplace in M365 Admin Center

Deploy:
 Sign in to: https//admin.powerplatform.microsoft.com
 Go to Environments. Click New, Provide Name, Type as Trial
 Add a Dataverse data store -> Yes
 Select appropriate Security Group
 Enable Dynamics 365 apps ->
 Select template related to trial license you have. such as Project Operation / Finance / SCM / Commerce
 Select Save to start the deployment
